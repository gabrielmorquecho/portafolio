Laboratorio 03: Querying the database type and version on Oracle

Progreso SQLi: 3 de 18 laboratorios completados.
Detalles Técnicos
  • Tipo: UNION-based SQLi
  • SGBD: Oracle Database
  • Objetivo: Extracción de Versión
Análisis del Ataque

A diferencia de otros sistemas, Oracle requiere que cada consulta SELECT tenga una cláusula FROM, utilizando la tabla dual para pruebas de estructura.

01. Determinación de Columnas (Oracle)

Identificamos el número de columnas y cuáles aceptan texto. En Oracle, usamos FROM dual para validar la estructura de la consulta UNION.

' UNION SELECT 'abc', 'def' FROM dual--

[EVIDENCIA: DETERMINACIÓN DE COLUMNAS]

02. Extracción de Versión del Banner

Utilizamos la tabla integrada v$version para consultar la información del sistema. Sustituimos el primer valor NULL por el parámetro BANNER.

' UNION SELECT BANNER, NULL FROM v$version--

[EVIDENCIA: PAYLOAD FINAL EN REPEATER]

03. Resultados: Versión Obtenida

Al procesar la solicitud, la página renderiza la cadena de texto con la versión exacta de Oracle Database, completando con éxito el desafío.
